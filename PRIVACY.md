# Privacy Notes

This kit is intentionally offline and merchant-controlled.

- It does not make network calls.
- It does not send data to AdPages, A1 Local, BigCommerce APIs, or third-party analytics tools.
- It does not include hidden tracking, pixels, beacons, enrichment, scraping, or fingerprinting.
- It does not collect credentials, customer account data, payment details, or order data.
- The Script Manager snippet stores campaign parameters in the shopper's browser only, using the merchant-configured storage key.
- Any checkout/order review values require merchant-created fields and should be disclosed in the merchant's own privacy policy where relevant.

Before production use, confirm consent, retention, checkout disclosure, and privacy-policy wording with the merchant's legal/privacy reviewer.
