# Publish Blockers

This directory is ready as a resource kit, not as a BigCommerce App Marketplace app.

Before claiming marketplace readiness, finish:

- BigCommerce app architecture decision: app, Script Manager tutorial, Stencil theme utility, or agency resource.
- BigCommerce developer account setup and final package identity.
- App OAuth/install flow if an app is required.
- Scope review if any BigCommerce API access is added.
- Production hosting, support URL, privacy URL, terms URL, screenshots, and listing copy.
- Merchant onboarding flow for checkout custom fields or order review process setup.
- QA on a development store with current Optimized One-Page Checkout behavior.
- Security/privacy review for retention, consent, and customer data handling.
- BigCommerce App Marketplace review and approval.

Do not describe this scaffold as approved, listed, official, or production-ready until those blockers are cleared.
